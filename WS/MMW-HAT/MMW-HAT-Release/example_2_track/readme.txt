Overview
This script demonstrates a radar tracking example. Simply run the script, and it will handle everything automatically.

How to Use

Run the script (in terminal/console):
python run_example_track.py

The GUI will start, and data will begin streaming.
The target will be displayed on the x-y plane in the GUI.

Known Limitations
It may take up to 10 seconds to start.
This script has been only tested on Raspberry Pi 5 with official power adapter.
If an SPI communication error occurs, data streaming may fail. Restart the script to resolve the issue.
