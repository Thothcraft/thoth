Overview
This script demonstrates UDP data streaming visualization. Simply run the script, and it will handle everything automatically.

How to Use

Run the script (in terminal/console):
python run_udp_streaming_vis.py

The following visualizations will be plotted:
Range-Doppler
Azimuth-Range
Azimuth-Doppler
Known Limitations

It may take up to 30 seconds to start.
This script has been only tested on Raspberry Pi 5 with official power adapter.
If an SPI communication error occurs, data streaming may fail. Restart the script to resolve the issue.
